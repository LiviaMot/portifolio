Projetos

Título do Projeto:

Calculadora de Gorjetas

Descrição:

Um projeto de estudo focado em aplicar os fundamentos do desenvolvimento front-end para criar uma ferramenta prática e funcional. A aplicação foi construída com HTML para a estrutura, CSS para a estilização e JavaScript para dar vida à calculadora. O desenvolvimento deste projeto foi essencial para praticar habilidades importantes como a manipulação do DOM para exibir os resultados, o gerenciamento de eventos para capturar o envio do formulário, e a implementação de lógicas de validação para garantir que os dados inseridos pelo usuário sejam corretos.
